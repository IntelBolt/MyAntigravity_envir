## [2026-02-07] Сессия: Решение проблемы с GitHub MCP

### Основные события:
1.  **Диагностика GitHub MCP**: Выявлена ошибка `exec: "docker": executable file not found`. Причина — попытка запуска MCP-сервера через Docker, который отсутствует в системе.
2.  **Решение**: Пользователь решил пока **отключить GitHub MCP**. 
3.  **LTM (Память)**: Сохранена инструкция на будущее — если GitHub MCP понадобится, его следует настраивать через `npx`, чтобы избежать зависимости от Docker.

## [2026-02-08] Сессия: Реализация Dashboard на Next.js

### Основные достижения:
1.  **Инфраструктура**: Успешно инициализирован Shadcn UI в папке `dashboard/`. Настроен Tailwind 4 с брендовыми цветами IntelBolt (Indigo/Slate).
2.  **Компоненты**: Созданы ключевые модули:
    - `Sidebar.tsx`: Навигация с логотипом и активными состояниями.
    - `KPICard.tsx`: Карточки метрик с трендами.
    - `RevenueChart.tsx`: Интерактивный график на базе Recharts.
    - `AIChat.tsx`: Интерфейс чата с агентом-аналитиком.
3.  **Сборка**: Основная страница `page.tsx` собрана и готова к работе.

### Технические сложности:
- **SWC Binary Error**: При попытке запуска `npm run dev` на диске `H:` (съёмный диск) возникает ошибка `@next/swc-win32-x64-msvc ... is not a valid Win32 application`. 
- **Решение**: Проблема, скорее всего, связана с ограничениями файловой системы на внешнем диске. Рекомендуется запускать проект с локального SSD или проверить права доступа к бинарным файлам в `node_modules`.

### Заметки для следующей сессии:
- Решить проблему с запуском бинарников (возможно, перенос на локальный диск).
- Настроить интеграцию с n8n API для динамических данных в чате и графиках.
