# CRM Management Skill

Этот навык предназначен для автоматизации рутинных задач в CRM-системах через API или веб-интерфейс.

## 1. Core Functions
- **Lead Processing**: Автоматическое создание, квалификация и распределение лидов.
- **Deal Management**: Обновление стадий сделок, добавление заметок и прикрепление документов.
- **Task Automation**: Создание напоминаний и задач для менеджеров на основе триггеров.
- **Reporting**: Извлечение данных для аналитических отчетов.

## 2. Interaction Guidelines
- **API First**: Всегда отдавать приоритет использованию официальных API (REST/GraphQL).
- **Data Integrity**: Перед обновлением данных проверять наличие обязательных полей (Email, Телефон, Stage ID).
- **Safety**: Никогда не удалять данные массово без явного подтверждения.

## 3. Workflow Examples
### Создание нового лида
1. Проверить дубликаты по email/телефону.
2. Создать запись в CRM.
3. Назначить ответственного менеджера.
4. Отправить уведомление в Slack/Telegram.

## 4. Specific CRM Support
- **Bitrix24**: Использовать вебхуки или REST API (`bx24` обертки).
- **amoCRM**: Использовать OAuth 2.0 и API v4.
- **HubSpot**: Использовать API ключи или OAuth.
